Sample code in Cold Fusion for RChilli JD Parsing Rest api version 3.0

This sample code help you to integrate with RChilli Api for resume parsing.
Sample code can be modified and used as per your need.

Sample code contain 2 file
ParseJD.cfml : this file can be used to parse the JD on desktop need to convert to base 64, you also need to set api credential in this file.
ParseJDText.cfml : this file can be used to parse jd text, and you need to set api credential too.
